# FrameFlow 写实图片生成提示词

共 36 条：12 个主题，每个主题 3 个独立场景。所有提示词的地点、人物数量、核心动作、时间、机位和关键道具均经过区分，避免重复素材。

## 统一要求

- `16:9` 横屏，建议 `1280×720` 或更高分辨率。
- 真实摄影质感、真实材质、自然光影和可信物理细节。
- 无可读文字、无字幕、无 Logo、无品牌标识、无平台水印、无边框。
- 人脸、手指、四肢、车辆、建筑和设备结构必须自然稳定。
- 避免插画、3D 渲染、塑料皮肤、过度磨皮、悬浮 UI 和过度霓虹效果。
- 批量生成时不要把上一张生成图作为下一张的参考图；模型支持随机种子时，每张使用不同种子。
